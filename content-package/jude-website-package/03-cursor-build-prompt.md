# Copy/Paste Prompt for Cursor AI Website Build

Build out my existing Jude single-page landing page into a full SEO-friendly website. Preserve the existing approved branding, colors, logo direction, and landing page visual feel. Expand the website using the content files in this package.

## Product
Jude is a wall-mounted AI companion for elderly adults focused on Home, Health, and Happiness. It helps seniors live safer, healthier, happier, and more independent lives at home while giving adult children and caregivers peace of mind.

## Design Direction
Use Apple.com as inspiration for structure: large hero sections, short confident headlines, simple CTAs, modular benefit tiles, spacious layout, strong product storytelling, and clean page hierarchy.

But Jude should be less sleek and more Southern charm: warm, trustworthy, family-first, homey, porch-light energy, soft sunlight, family kitchen, cozy interiors, wood tones, framed device, morning coffee, pets, and family photos.

## Build Requirements
1. Add a full homepage using `02-homepage-full.md`.
2. Create standalone SEO pages from every file in `/pages/`.
3. These are not blog posts visually. They should be permanent educational landing pages.
4. Use clean URLs based on each page slug.
5. Add internal links between related pages.
6. Add FAQ schema for the FAQ sections.
7. Add meta title and meta description from each file.
8. Add a global CTA at the bottom of each page.
9. Add a footer with page groups for Home, Health, Happiness, Family, and Why Jude.
10. Make the pages mobile responsive.
11. Use large type, strong spacing, warm photography/image blocks, and simple CTA buttons.
12. Do not make medical claims. Use “supports,” “helps,” “reminds,” and “can integrate with compatible devices.”
13. Do not make Jude feel like surveillance. Use privacy-first and dignity-first language.
