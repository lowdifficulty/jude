# Jude Website Content Package

This package expands the current one-page Jude landing page into a full SEO-friendly website.

## Included
- `00-website-architecture.md` — sitemap, URL structure, page template, and content strategy.
- `01-global-copy-and-design-system.md` — brand voice, design direction, CTA language, and compliance wording.
- `02-homepage-full.md` — expanded homepage copy.
- `03-cursor-build-prompt.md` — copy/paste implementation prompt for Cursor AI.
- `04-seo-page-index.csv` — page list with slugs, keywords, descriptions, and audiences.
- `navigation.json` — suggested navigation and footer grouping.
- `content-model.json` — structured brand and page metadata.
- `/pages/` — 37 SEO landing pages in Markdown.
- `/pages-html/` — simple HTML previews of every page.
- `/components/` — reusable CTA, page template, and schema notes.

## Strategic Direction
The website should feel like a clean product site with simple, confident, Apple-inspired product storytelling, but warmer and less sterile. Jude should feel like a helpful presence in a Southern family home: calm, trustworthy, charming, useful, and loved.

## Implementation Notes
- These pages are SEO landing pages, not dated blog posts.
- Each page should have permanent navigation and internal links.
- Use FAQ schema on every page.
- Avoid medical claims. Use supportive language.
- Avoid surveillance language. Use trust, consent, and family peace-of-mind language.
