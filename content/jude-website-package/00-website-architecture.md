# Jude Full Website Architecture

## Goal
Build a full SEO-friendly website from the existing single-page landing page. Keep the approved branding, expand the story, and create standalone benefit pages that rank for elder care, aging-in-place, AI companion, smart home, family caregiver, and senior safety searches.

## Design Influence
Apple-inspired structure: short hero copy, bold product statements, clear section hierarchy, simple CTAs, large visual moments, modular tiles, and benefit-led pages.

Jude difference: less sleek, more Southern charm. Warm, homey, trustworthy, family-first, porch-light energy.

## Page Template
Each SEO benefit page should include:

1. Meta title
2. Meta description
3. Primary keyword
4. Hero headline
5. Emotional subheadline
6. Primary CTA
7. Short benefit intro
8. Three core sections
9. Benefit bullet list
10. Use-case cards
11. FAQ section
12. Internal links to related pages
13. Final CTA

## Global CTA Options
- Join the Jude waitlist
- See how Jude helps families
- Bring Jude home
- Learn how Jude supports aging in place
- Talk to Jude
- Get updates

## Global Copy Rules
- Never make Jude sound like surveillance.
- Avoid overpromising medical outcomes.
- Use “supports,” “helps,” “reminds,” “coordinates,” and “encourages.”
- Focus on dignity, independence, calm, comfort, safety, family, and warmth.
- Use clean product copy with warm human language.
