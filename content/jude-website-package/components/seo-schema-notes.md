# SEO Schema Notes

Use these schema types where appropriate:

- Organization schema for Jude brand.
- WebSite schema for the main site.
- WebPage schema on each landing page.
- FAQPage schema for FAQ sections.
- BreadcrumbList schema for page hierarchy.
- Product schema only when product details, pricing, and availability are finalized.

Do not use MedicalBusiness or medical claims unless the business model and compliance review support it.
